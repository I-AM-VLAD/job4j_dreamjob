# DreamJob

## Описание проекта

DreamJob — это веб-приложение для взаимодействия кандидатов и кадровиков.

Платформа позволяет:
- кандидатам создавать и публиковать свои резюме
- кадровикам размещать вакансии
- управлять данными (создание, редактирование, удаление)

---

## Функциональность

### Кандидаты
- Просмотр списка кандидатов
- Создание нового кандидата
- Редактирование данных кандидата
- Удаление кандидата
- Загрузка файлов (например, резюме)

### Вакансии
- Просмотр списка вакансий
- Создание вакансии
- Редактирование вакансии
- Удаление вакансии
- Привязка вакансии к городу
- Загрузка файлов

---

## Архитектура

Приложение построено по MVC:

- **Controller** — обработка HTTP-запросов  
- **Service** — бизнес-логика  
- **Repository (DAO)** — работа с БД  
- **View (Thymeleaf)** — отображение  

---

## Запуск проекта

1. Установить PostgreSQL  
2. Создать базу данных:
CREATE DATABASE dreamjob;

## Запустить проект:

mvn spring-boot:run
